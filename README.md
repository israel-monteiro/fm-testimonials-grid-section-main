# Testimonials Grid Layout (Mobile-First) | Frontend Mentor

## 📌 Overview

This project is a solution to the Testimonials Grid Section challenge from Frontend Mentor. The goal was to build a fully responsive testimonials layout using modern CSS techniques such as Grid and Flexbox, following a mobile-first approach to ensure optimal user experience across different screen sizes.

## 🚀 How to run the project

Clone the repository:

    git clone https://github.com/israel-monteiro/fm-testimonials-grid-section-main.git

Navigate to the project folder:

    cd fm-testimonials-grid-section-main

Open the `index.html` file in your browser.
